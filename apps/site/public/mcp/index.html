IMPT × Claude (MCP)

Let Claude book hotels.

Add the IMPT MCP server to Claude Desktop or Claude Code. The model gains four tools — search_hotels, get_quote, get_deeplink, track — and produces canonical booking links your readers can complete in one click.

Install in Claude Desktop

Add this to your claude_desktop_config.json:

{
  "mcpServers": {
    "impt": {
      "command": "npx",
      "args": ["-y", "@impt/mcp-server"],
      "env": { "IMPT_PARTNER_KEY": "YOUR_KEY" }
    }
  }
}

Install in Claude Code

claude mcp add impt npx -y @impt/mcp-server -e IMPT_PARTNER_KEY=YOUR_KEY

HTTP transport (custom clients)

SSE: https://swarm.impt.io/api/mcp/sse · Streamable HTTP: https://swarm.impt.io/api/mcp/http
